Express GraphQL

Installation of GraphQL

Start by including express, express-grapqhql, and graphql at the top of the script.

Example: 

const express = require('express')
const graphqlHTTP = require('express-graphql')
const {buildSchema} = require('graphql')

Build a Schema and Root

Simply pass a string to buildSchema and define a response in root.

Example: 

const schema = buildSchema(`
  type Query {
    hello: String
  }
`)
const root = {
	hello: () => {
		return "Hello World"
	}
}

Set up Route

Make a graphql route in express and pass graphqlHTTP where the response and request would be.

app.use('/graphql', graphqlHTTP({
  schema: schema,
  rootValue: root,
  graphiql: true // enables the default interface
}))
Important! Do not forget to set up the default express app.
